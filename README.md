# Kanuorie Tasks Manager (Kanban Productivity App)

A full-stack Kanban board application that helps users organize tasks efficiently with drag-and-drop functionality, authentication, and a clean modern UI.

## ✨ Features

- 🔐 User Authentication (Register & Login)
- 📝 Create, Edit, Delete Tasks
- 📌 Drag & Drop Kanban Board
- 🔄 Task Status Management (Backlog → Progress → Done)
- 🔍 Search & Filter Tasks
- 📱 Responsive UI
- 🎯 Clean SaaS-style Landing Page


## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router
- CSS (Custom UI)

### Backend
- Node.js
- Express.js
- SQLite

### Authentication
- JWT (JSON Web Tokens)
- bcrypt (Password hashing)

## Usage:

-Register a new account,
-Login,
-Create tasks
-Drag tasks between columns
-Edit or delete tasks
